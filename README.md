## Building the resources

This project was built using CMake for convenience as there are lots of executables.

### Cmake

```sh
mkdir -p build
cd build
cmake ..
make
```
## GCC

Needs to be done one by one.
Refer to below example:
```sh
cc -I ./include src/p1/client.c -o client.out
```
